Help

Meeting Notes keeps meeting context, follow-up actions, deadlines, and tags together in one workspace.

How to use the workspace

  • Use the note editor to capture narrative context and action items separately.
  • Action items appear both inside the note card and in the General Todo card unless the note is archived.
  • Search and filters help narrow notes by text, date, status, and selected tags.
  • Archive notes to keep them for reference while removing their follow-up work from the active dashboard.

Keyboard shortcuts

Ctrl/Cmd + N New note
Ctrl/Cmd + S Save note
Ctrl/Cmd + F Open search
Ctrl/Cmd + E Toggle light and dark mode
/ Focus search
Esc Close the active dialog
