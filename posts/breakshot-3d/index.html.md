Breakshot 3D is a standalone browser pool game with local play, CPU matches, and a support layer of rules and guides.

The product is not just the game loop. It also tests whether useful search pages around aiming, spin, rules, and play modes can help a browser game become discoverable.

## What It Tests

- Browser-based 3D gameplay that loads quickly enough to be approachable.
- A rules and guide layer that supports players outside the main game canvas.
- The boundary between a game prototype and a standalone product domain.
